# 公文格式核稿系统

基于 GB/T 9704 标准的公文格式自动检测与修正工具。上传 `.docx` 公文文件，系统自动识别文档结构，检测格式违规项，并一键生成修正后的文档。

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务器
python app.py
```

服务器运行在 `http://localhost:8000`。

## 功能

- 上传 `.docx` 公文文件，自动解析文档结构（标题、各级标题、正文、附件、落款等）
- 检测格式问题：字体、字号、行距、缩进、对齐、页边距等
- 左右栏预览：当前格式 vs 修正后格式
- 一键生成修正版文档并下载
- 支持批量上传与批量处理

## 格式规范

参照 GB/T 9704《党政机关公文格式》标准：

| 元素 | 字体 | 字号 | 缩进 |
|------|------|------|------|
| 标题 | 方正小标宋_GBK | 22pt (2号) | 无 |
| 一级标题 | 黑体 | 16pt (3号) | 1.13cm |
| 二级标题 | 楷体_GB2312 | 16pt (3号) | 1.13cm |
| 三级标题/正文 | 仿宋_GB2312 | 16pt (3号) | 1.13cm |
| 落款/日期 | 仿宋_GB2312 | 16pt (3号) | 无 |

- 固定行距：28 磅
- 页边距：上 37mm / 下 35mm / 左 28mm / 右 26mm

## 字体要求

系统需要以下字体（`fonts/` 目录提供下载）：

- 方正小标宋_GBK（公文标题）
- 仿宋_GB2312（正文/三级标题/落款）
- 楷体_GB2312（二级标题/附注）
- 黑体（一级标题，系统自带）
